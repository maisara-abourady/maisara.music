# Hero redesign brief

Status: **Awaiting inspiration via Google Stitch MCP — do not implement yet.**

## Goal

Redesign the hero section *creatively* — but stay 100% inside the existing aesthetic and design system already established across the site.

## Hard constraints (design system carries over)

- Background: deep navy near-black (`--bg #0a0e1a`, `--bg-alt #0c1120`).
- Single accent: cool atmospheric blue (`--accent #74BFFF`, dim `#77b0d1`, glow `rgba(116,191,255,0.12)`).
- Type: Cormorant Garamond (light italics for emotional headlines), Poppins (UI, body, eyebrows).
- Eyebrow style: tiny uppercase, ~0.38em letterspacing, accent-dim color, leading hairline.
- Hairlines (1px), faint grain overlay, soft radial glows from edges, generous negative space.
- Restraint as language — nothing loud, nothing fast.
- Portrait of MAISARA (`assets/hero-bg.png`) remains the visual anchor of the hero. Free to reposition / crop / scale / mask / layer — but the lighting world (single soft source upper-right, navy haze, dust particles) is the lighting logic for the hero.

## What changes

Replace the current bottom-nav-on-hero pattern. The hero itself becomes the navigation entry point for the site, with **two primary destinations elevated as the main CTAs of the entire website**:

1. **Listening Room** — at-home / non-dancefloor mixes for deep listening, headphones, late nights, journeys.
2. **Warm-up Sessions** — long-form, patient, mood-shaping sets played in the opening hours of a night, before the headliner. The heart of his identity.

These two CTAs must feel like *the* point of the site, not a list of links. Everything else (Philosophy, Sound/Style, Projects, Bookings) is secondary navigation reachable through the sticky nav post-hero or via a quieter affordance.

Naming note: in earlier brainstorming these were called "Warm-up Sets" and "Listening Sets" — the locked names are now **Listening Room** and **Warm-up Sessions**. Update the rest of the site to match when we implement.

## Creative latitude

- Asymmetry welcome. The two CTAs do not have to be twin buttons — they can have different visual weight, different positions, different rhythms, as long as both feel like primary destinations.
- The portrait can move. It does not have to be centered or full-bleed.
- Unexpected scroll behaviors, oversized typography, vertical type, waveform motifs, timeline/tracklist metaphors, marquee tickers — all on the table.
- Surprise on structure; honor the feeling.

## Process

1. User will install Google Stitch MCP.
2. User will share an inspiration source through it.
3. Only then do we design the hero. **Do not start before that.**
